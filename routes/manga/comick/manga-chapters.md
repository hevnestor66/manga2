# Manga Chapters

### Request

```http
https://jimov.herokuapp.com/manga/comick/chapter/{chapter-id}?lang={language}
```

* `chapter-id` - ID of the manga chapter to obtain the pictures
* `language`-  Language in which the images are displayed: br -> Brazil, en -> United States, es -> Spanish, fr -> France, ru -> Russia

### Example

```http
https://jimov.herokuapp.com/manga/comick/chapter/DJlpLGKT-dansou-no-partner-7?lang=en
```
