# Anime Info

### Request

```http
https://jimov.herokuapp.com/anime/animelatinohd/name/{name}
```

* `:name`⁣ – The name of the anime to retrieve information about.

### Example

```http
https://jimov.herokuapp.com/anime/animelatinohd/name/wonder-egg-priority
```
