# Anime Info

### Request

```http
https://jimov.herokuapp.com/anime/tioanime/name/{name}
```

* `:name`⁣ – The name of the anime to retrieve information about.

### Example

```http
https://jimov.herokuapp.com/anime/tioanime/name/wonder-egg-priority
```
